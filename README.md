# AI Churn Prediction Dashboard

A professional full-stack AI-powered customer churn prediction dashboard built using React, Flask, and Machine Learning.

## Features
- Customer churn prediction
- Interactive dashboard
- Data visualization
- Machine learning integration
- Responsive UI

## Tech Stack
### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Flask
- Python
- Scikit-learn

### Data Science
- Pandas
- NumPy
- Matplotlib

## Project Structure

AI-Churn-Prediction-Dashboard/
│
├── frontend/
├── backend/
├── dataset/
└── screenshots/

## Installation

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
